# SpectroGhost
SpectroGhost is a fast, browser-based spectrogram viewer for hunting hidden audio messages. Load MP3/WAV, choose FFT size and hop, log/linear scale, min/max Hz and dynamic range. Zoom, pan, and scroll through time; rulers track cursor. High-res rendering with Audacity-like colormap. Export PNGâ€”no backend required. Works offline via Web Audio STFT.
